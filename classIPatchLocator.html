Detailed Description

template<class X, class Y>
class IPatchLocator< X, Y >

Base class to locate on which patch a given physical coordinate is.

The physical coordinates are supposed to be global and the logical coordinates are supposed to be locally defined.

XFirst physical coordinate.
YSecond physical coordinate.
