Detailed Description

template<class PoissonSolver, class AdvectionX, class AdvectionY>
class PredCorrRK2XY< PoissonSolver, AdvectionX, AdvectionY >

Predictor-corrector based on RK2 for the guiding-center model.

It solves in time the following guiding-center equations system:

  • \( -\Delta \phi = f \),
  • \( E = - \nabla \phi \),
  • \( \partial_t f - E_y \partial_x f + E_x \partial_y f= 0\).

This method is mainly a Runge-Kutta 2 method:

for \( n \geq 0 \),

First, it advects on a half time step:

  • 1./2. From \(f^n\), it computes \(E^n\) with a FFTPoissonSolver;
  • 3. From \(f^n\) and \(E^n\), it computes \(f^{n+1/2}\) with a BslAdvection1D on \(\frac{dt}{2}\);

Secondly, it advects on a full time step:

  • 4./5. From \(f^{n+1/2}\), it computes \(E^{n+1/2}\) with a FFTPoissonSolver;
  • 6. From \(f^n\) and \(E^{n+1/2}\), it computes \(f^{n+1}\) with a BslAdvectionRP on \(dt\).
