Detailed Description

template<class AdvectionField, class FunctionToBeAdvected>
class AdvectionSimulation< AdvectionField, FunctionToBeAdvected >

Base class for the tests simulation of the 2D polar advection operator.

The simulations are:

  • TranslationSimulation,
  • RotationSimulation,
  • DecentredRotationSimulation.
