Detailed Description

template<class Spline1DBuilder, class Spline1DEvaluator>
class Spline1DPartialDerivativeCreator< Spline1DBuilder, Spline1DEvaluator >

A class which stores information necessary to create a pointer to an instance of the Spline1DPartialDerivative class.

This class allows an instance of the Spline1DPartialDerivative class to be instantiated where necessary. Typically, the Spline1DPartialDerivativeCreator is instantiated in the initialisation of the simulation, and the corresponding Spline1DPartialDerivative object is instantiated where computing partial derivatives is required.

Spline1DEvaluatorA 1D spline evaluator.
