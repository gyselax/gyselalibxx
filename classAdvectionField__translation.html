Detailed Description

Advection field for the tests of the 2D polar advection operator.

The test advection field for a translation in the physical domain is given by :

\( A(x,y) = [v_x, v_y]\).

with \( v_x \) and \( v_y \) constants.

The characteristic feet are then given by

  • \( X(t + dt) = X(t) + dt v_x \),
  • \( Y(t + dt) = Y(t) + dt v_y \).
