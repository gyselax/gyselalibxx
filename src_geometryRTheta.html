Geometry (r, theta)

The geometryRTheta folder contains all the code describing methods which are specific to a 2D curvilinear geometry containing a singular point. It is broken up into the following sub-folders:

  • Advection operator - Code describing advection operator and time integration methods used on 2D polar domain.
  • Advection Field finder - Code describing computation of the advection field given to the advection operator.
  • src_geometryRTheta_geometry - All the dimension tags used for a gyrokinetic semi-Lagrangian simulation in a curvilinear geometry.
  • Initialization - Initializes the simulations on a 2D polar domain.
  • Spline interpolator in polar coordinates - Code describing interpolation methods on 2D polar domain.
  • Polar Poisson solver - Code describing the polar Poisson solver.
  • Predictor-corrector methods - The methods to solve in time the equations system.
