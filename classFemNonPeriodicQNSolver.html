Detailed Description

An operator which solves the Quasi-Neutrality equation using Finite Elements on a non-periodic domain.

An operator which solves the Quasi-Neutrality equation: \( - \frac{d^2 \phi}{dx^2} = \rho \) using Finite Elements on a non-periodic domain. This solver uses spline basis elements and imposes Dirichlet boundary conditions.
