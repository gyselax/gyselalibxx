Detailed Description

template<class Mapping>
class FunctionToBeAdvected_cos_4_elipse< Mapping >

Test function for the 2D polar advection operator.

The test function is given in Edoardo Zoni's article (https://doi.org/10.1016/j.jcp.2019.108889):

\( f(x, y) = \frac{1}{2}\left( G(r_1(x,y)) + G(r_2(x,y))\right)\)

with

  • \(G(r) = \cos\left(\frac{\pi r}{2 a}\right)^4 * 1_{r<a}(r)\),
  • \(r_1(x, y) = \sqrt{(x-x_0)^2 + 8(y-y_0)^2}\),
  • \(r_2(x, y) = \sqrt{8(x-x_0)^2 + (y-y_0)^2}\).
