Detailed Description

The ND Lagrange polynomial is defined as: \(\[ L(x_1, \ldots, x_N) = \sum_{i_1=0}^{d_1}\dots\sum_{i_N=0}^{d_N} \prod_{j=1}^N f(x_1, \ldots, x_N) l_{j,i_j}(x_j) \]\) The evaluation is recursive: \(\[ L(x_1, \ldots, x_N) = \sum_{i_1=0}^{d_1} l_{1,i_1}(x_1) L(x_2, \dots, x_N) \]\)

The recursion terminates when only one evaluator remains: in that case the tail type is the LagrangeEvaluator itself and the 1D evaluation (including its configured extrapolation rules) is invoked directly.

Template parameters:

  • HeadEvaluator The 1D LagrangeEvaluator for the first dimension.
  • Evaluators1D The 1D LagrangeEvaluators for the remaining dimensions.
