Detailed Description

DerivField stores the values and the derivatives of a function. The inputs of the SplineBuilder2D are on different layouts than the fields we can get from the DerivField. SplineBuliderDerivField2D allows to directly apply a stored SplineBuilder2D to a DerivField by copying data in fields with correct layout.

Implemented only for 2D case.
