Detailed Description

This class allows an instance of the SplineInterpolator class where necessary. This allows the memory allocated in the private members of the SplineInterpolator to be freed when the object is not in use. These objects are: m_coefs, m_derivs_min_alloc, m_derivs_max_alloc.
