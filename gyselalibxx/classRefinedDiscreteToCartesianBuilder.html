Class RefinedDiscreteToCartesianBuilder

template <class X, class Y, class SplineBuilder, class SplineEvaluator, int ncells_r, int ncells_theta>

ClassList > RefinedDiscreteToCartesianBuilder

A class to create a DiscreteToCartesian instance from an analytical mapping. This class creates an instance which uses more refined splines than the provided builder and evaluator. This class creates and stores splines memory spaces describing the analytical mapping. The discrete mapping is then created using the splines without copying data.More...
