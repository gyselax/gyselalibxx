Gyselalib++
Script to automise :

- the creation of the initial restart file with the python script <tt>init_distribution.py</tt>

- the creation of the input YAML file required as input of the C++ simulation <tt>testcollision</tt>

For instance, the following command

./testcollisions.sh input_params_ion_1x1x1x128x64.yaml

create the folder <tt>D_INPUT_PARAMS_ION_1X1X1X128X64.YAML</tt> containing:

- <tt>GysX_rst_00000.h5</tt> : output of the python script <tt>init_distribution.py</tt>

- <tt>GysX_rst_00001.h5</tt> : output of the C++ collision executable

- <tt>coll_ref.yml</tt> : input for C++ collision executable automatically created by the bash script <tt>testcollision.sh</tt>

- <tt>diff_f_vpar_mu_itor1eq0_itor2eq0_itor3eq0_ispeq0.png</tt> : output figure to compare the results between <tt>GysX_rst_00000.h5</tt> and <tt>GysX_rst_00001.h5</tt>
