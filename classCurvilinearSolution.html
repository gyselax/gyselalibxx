Detailed Description

template<class CurvilinearToCartesian>
class CurvilinearSolution< CurvilinearToCartesian >

Define a curvilinear solution of the Poisson equation.

The solution is given by

  • \( \phi(x, y) = C r(x,y)^6 (r(x,y) -1)^6 \cos(m\theta) \),

with \(C = 2^{12}1e-4 \) and \( m = 11 \).
