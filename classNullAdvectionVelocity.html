Detailed Description

template<class IdxRangeFdistribu, class IdxRangeSpatial>
class NullAdvectionVelocity< IdxRangeFdistribu, IdxRangeSpatial >

This is a class which imitates a velocity advection.

It inherits from IAdvectionV and can be used as an advection operator but does not actually modify the distribution function. This can be useful for debugging purposes.
