Instantiate a PolarSplineEvaluator with boundary evaluation conditions.

Instantiate a PolarSplineEvaluator by specifying how points lying outside the domain should be evaluated. The domain is the domain on which the polar splines are defined.

Parameters
[in]outer_bcA class containing an operator which can be called to provide a boundary value to evaluate a point lying outside the domain.
