Detailed Description

template<class DDimI, BCond BcMin, BCond BcMax, class... DDim>
class PreallocatableLagrangeInterpolatorBatched< DDimI, BcMin, BcMax, DDim >

A class which stores information necessary to create an instance of the LagrangeInterpolatorBatched class.

This class allows an instance of the LagrangeInterpolatorBatched class where necessary. This allows the memory allocated in the private members of the InterpolatorBatched to be freed when the object is not in use.
