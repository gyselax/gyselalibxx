Detailed Description

template<class X, class Y, class SplineRThetaBuilder, class SplineRThetaEvaluator, int Nr, int Nt>
class RefinedDiscreteToCartesian< X, Y, SplineRThetaBuilder, SplineRThetaEvaluator, Nr, Nt >

A class for describing refined discrete 2D mappings from the logical domain to the physical domain.

The mapping is a DiscreteToCartesian mapping but built on a different grid. To be independent of errors due to the use of a discrete mapping, it could be interesting to work with a discrete mapping built on a refined grid.

Here, we wrap a DiscreteToCartesian mapping defined on a refined grid in a RefinedDiscreteToCartesian mapping. The creation of the refined grid of size Nr x Nt ( \(= N_r \times N_\theta \)) is made in the class.
