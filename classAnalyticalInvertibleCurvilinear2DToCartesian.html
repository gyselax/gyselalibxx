Detailed Description

template<class X, class Y, class R, class Theta>
class AnalyticalInvertibleCurvilinear2DToCartesian< X, Y, R, Theta >

A class for describing analytical invertible curvilinear 2D mappings from the logical domain to the physical domain.

We define a class of mapping not costly invertible (because analyticaly invertible).

pure virtual

Compute the physical coordinates from the logical coordinates.

The mapping from the logical domain \( (r,\theta) \) to the physical domain \( (x,y) \) must always be well defined. However the inverse mapping is not always well defined especially at the center point or can be costly to inverse.
