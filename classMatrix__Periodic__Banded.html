Detailed Description

A class representing a periodic banded matrix.

A periodic banded matrix is like a banded matrix but additionally contains non- zero values in the corners. I.e it has the following sparsity pattern: 

* * *                   * *
* * * *                   *
* * * * *
  * * * * *

          ...

                  * * * * *
*                   * * * *
* *                   * * *

◆ calculate_delta_to_factorise()

virtual void Matrix_Periodic_Banded::calculate_delta_to_factorise ( )
overrideprotectedvirtual

Calculate the contents of the dense matrix \( delta' \) that will be factorised.

This is the element \( delta' \) of the blockwise LU decomposition of the matrix. 

L = |   Q    |    0   |
    | lambda | delta' |

It is defined as:

\( \delta' = \delta - \lambda \beta \)

With \( \beta \) the solution of the equation:

\( Q \beta = \gamma \)

which should already be stored in the variable Abm_1_gamma when this method is called.
