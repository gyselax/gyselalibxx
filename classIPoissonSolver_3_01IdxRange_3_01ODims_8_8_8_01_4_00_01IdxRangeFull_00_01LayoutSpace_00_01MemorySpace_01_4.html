Detailed Description

template<class... ODims, class IdxRangeFull, class LayoutSpace, class MemorySpace>
class IPoissonSolver< IdxRange< ODims... >, IdxRangeFull, LayoutSpace, MemorySpace >

An abstract class from which a Poisson solver can inherit.

Classes inheriting from this must implement a way to solve the following equation: \( -\Delta \phi = \rho \)

Template Parameters
IdxRangeLaplacianThe index range on which the equation is defined.
IdxRangeFullThe index range on which the operator() acts. This is equal to the IdxRangeLaplacian plus any batched dimensions.
LayoutSpaceThe layout space of the Fields passed to operator().
MemorySpaceThe space (CPU/GPU) where the Fields passed to operator() are saved.
