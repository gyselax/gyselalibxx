Detailed Description

Advection field for the tests of the 2D polar advection operator.

The test advection field for a rotation in the physical index range is given by :

\( A(x,y) = J_{\mathcal{F}}[v_r, v_\theta]\).

with \( v_r \) and \( v_\theta \) constants and \( \mathcal{F}\) the circular mapping.

The characteristic feet are then given by

  • \( (X(t + dt), Y(t + dt)) = \mathcal{F} (R(t + dt), \Theta(t + dt))\),
  • with
    • \( R(t + dt) = R(t) + dt v_r \),
    • \( \Theta(t + dt) = \Theta(t) + dt v_\theta \),
    • and \( (R(t), \Theta(t)) = \mathcal{F}^{-1} (X(t), Y(t))\).
