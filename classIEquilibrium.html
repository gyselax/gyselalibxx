Detailed Description

An abstract class for initialising a distribution function in (species,vpar,mu).

An abstract class for initialising the equilibrium state of the distribution function.

An abstract class for initialising a distribution function that does not depend on space.

The equilibrium state does not depend on spatial dimensions.
