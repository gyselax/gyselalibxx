Detailed Description

template<class SplineBuilder2DCache, class SplineEvaluator2D, class DerivativeDimension>
class Spline2DPartialDerivativeCreator< SplineBuilder2DCache, SplineEvaluator2D, DerivativeDimension >

A class which stores information necessary to create a pointer to an instance of the Spline2DPartialDerivative class.

This class allows an instance of the Spline2DPartialDerivative class to be instantiated where necessary. Typically, the Spline2DPartialDerivativeCreator is instantiated in the initialisation of the simulation, and the corresponding Spline2DPartialDerivative object is instantiated where computing partial derivatives is required.
