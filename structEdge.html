Detailed Description

template<class IDim>
struct Edge< IDim >

Define an edge of a given patch.

An edge is defined by a patch index, a dimension and an extremity. For example, in the patch defined on logical domain \( [a_x, b_x]\times[a_y, b_y] \),

  • the edge IDimX, BACK refers to the set \( [a_x, b_x]\times\{b_y\} \),
  • and the edge IDimX, FRONT refers to the set \( [a_x, b_x]\times\{a_y\} \).

Here, the domain given as input corresponds to \( [a_x, b_x] \) in both case.
