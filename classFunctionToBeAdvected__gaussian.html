Detailed Description

template<class Mapping>
class FunctionToBeAdvected_gaussian< Mapping >

Test function for the 2D polar advection operator.

The test function is a Gaussian centred in \( (x_0, y_0 \)):

\( f(x, y) = C \exp\left( - \frac{(x - x_0)^2}{2\sigma_x^2} - \frac{(y - y_0)^2}{2\sigma_y^2} \right)\)
