Detailed Description

template<class GridSp, class GridVpar, class GridMu>
class CollisionConfiguration< GridSp, GridVpar, GridMu >

Class to collect information to initialise the collision operator for a SpVparMu geometry.

NOTE: Thanks to C++17 template constructor argument type deduction, we do not need to give the 3 template arguments. Giving a correct index_range_fdistribution is sufficient.
