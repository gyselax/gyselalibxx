Detailed Description

template<class ElectrostaticalPotentialSimulation, class FunctionToBeAdvected, class AdvectionField>
class AdvectionFieldSimulation< ElectrostaticalPotentialSimulation, FunctionToBeAdvected, AdvectionField >

Base class for the tests simulation of the 2D polar advection operator.

The simulations are:

  • TranslationAdvectionFieldSimulation,
  • RotationAdvectionFieldSimulation,
  • DecentredRotationAdvectionFieldSimulation.
