Detailed Description

template<class ElementType, class... DDims, int NDerivs, class Allocator>
class DerivField< ElementType, ddc::DiscreteDomain< DDims... >, NDerivs, Allocator >

A class which holds a chunk of memory describing a field and its derivatives.

The values of the field and the derivatives may be defined on different domains, but the underlying mesh must be the same for both.

Template Parameters
ElementTypeThe type of the elements inside the chunks.
ddc::DiscreteDomain<DDims...>The domain on which the internal chunks are defined. This domain is the physical domain on which the values are defined combined with the domain of the derivatives of interest (e.g. ddc::DiscreteDomain<Deriv<IDimX>, IDimX, IDimY>).
NDerivsThe number of derivatives which are defined in the dimensions where derivatives appear.
AllocatorThe Allocator which is used to build the Chunk. This provides information about the memory space where the data will be saved.
