Detailed Description

template<class... ODims, class FullIdxRange, class LayoutSpace, class MemorySpace>
class IPoissonSolver< IdxRange< ODims... >, FullIdxRange, LayoutSpace, MemorySpace >

An abstract class from which a Poisson solver can inherit.

Classes inheriting from this must implement a way to solve the following equation: \( -\Delta \phi = \rho \)

Template Parameters
LaplacianIdxRangeThe index range on which the equation is defined.
FullIdxRangeThe index range on which the operator() acts. This is equal to the LaplacianIdxRange plus any batched dimensions.
LayoutSpaceThe layout space of the ChunkSpans passed to operator().
MemorySpaceThe space (CPU/GPU) where the ChunkSpans passed to operator() are saved.
