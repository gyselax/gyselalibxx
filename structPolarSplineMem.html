inlineexplicit

A constructor for the PolarSplineMem.

A constructor for the PolarSplineMem which takes a 2D domain of bsplines. This domain is then split into the domains which are relevant for a polar bspline constructed from these 2D bsplines. These new domains are then used to create the chunks which define the spline.

Parameters
domainA 2D domain of bsplines.
