Detailed Description

template<class... GridPDEDim1D, class IdxRangeFull, class ExecSpace, class LayoutSpace>
class FFTPoissonSolver< IdxRange< GridPDEDim1D... >, IdxRangeFull, ExecSpace, LayoutSpace >

A class to solve the following equation: \( -\Delta \phi = \rho \) using a Fourier transform.

The implementation of this class can be found at FFTPoissonSolver< IdxRange<GridPDEDim1D...>, IdxRangeFull, ExecSpace, LayoutSpace >.

Template Parameters
IdxRangeLaplacianThe index range on which the equation is defined.
IdxRangeFullThe index range on which the operator() acts. This is equal to the IdxRangeLaplacian plus any batched dimensions.
ExecSpaceThe space (CPU/GPU) where the calculations will take place.
LayoutSpaceThe layout space of the Fields passed to operator().
